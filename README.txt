Для запуска файла необходимо:
1. Импортировать дамп базы данных в свою СУБД в базу с именем database.
2. Ввести данные для подключения к базе данных в файл project/includes/config.php

        'server' => 'localhost', - адрес сервера СУБД
        'user' => 'root', - имя пользователя для входа
        'password' => '', - пароль для входа
        'name' => 'database' - имя базы данных

3. Переместить директорию проекта "project" на сервер.
4. Запустить файл index.php .
